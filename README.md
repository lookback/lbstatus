## Development

Requirements:

- [Deno](https://deno.land/#installation)

Install on Mac/Linux with:

```bash
curl -fsSL https://deno.land/install.sh | sh
```

### Running

```bash
deno task run [args]
```

### Compile binaries

```bash
deno task compile
```
Will output binaries in `target` dir.
